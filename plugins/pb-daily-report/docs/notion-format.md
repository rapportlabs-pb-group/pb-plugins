# Notion 테이블 필수 형식 (v6.4)

CLAUDE.md에서 분리된 상세 형식 가이드입니다.

## 기본 형식

```html
<table>
<tr>
<td>헤더1</td>
<td>헤더2</td>
</tr>
<tr>
<td>값1</td>
<td>값2</td>
</tr>
</table>
```

## 규칙 테이블

| 규칙 | 올바른 형식 | 잘못된 형식 |
|------|-------------|-------------|
| 태그 속성 | `<table>` | `<table header-row="true">` |
| td 배치 | 각 `<td>` 별도 줄 | `<td>A</td><td>B</td>` 인라인 |
| 구분선 행 | (사용 안함) | `:---` 행 추가 → 보이는 행으로 렌더링됨 |

## 중요 사항

- 모든 `<td>` 태그는 별도 줄에 배치
- 속성 없는 순수 `<table>` 태그 사용
- `:---` 구분선 행 사용 금지 (Notion HTML 테이블에서 데이터 행으로 보임)
