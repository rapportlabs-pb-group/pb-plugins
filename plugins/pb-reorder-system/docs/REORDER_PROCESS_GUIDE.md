# PB 리오더 시스템 가이드

## 1. 개요

리오더는 잘 팔리는 상품의 재고가 부족해지기 전에 **미리 추가 발주**하는 시스템입니다.

| 항목 | 내용 |
|------|------|
| 실행 시간 | 매일 오전 10시 (평일만) |
| 대상 브랜드 | 노어, 퀸즈셀렉션, 베르다, 지재, 다나앤페타 |
| 핵심 원칙 | **과재고 방지 > 품절 방지** |

---

## 2. 발주량 계산

### 2.1 기본 공식

> **발주량 = 예상 판매량 - 보유량**

### 2.2 계산 예시

| 구분 | 설명 | 예시 |
|------|------|------|
| **(A) 예상 판매량** | 앞으로 2~3주간 팔릴 양 | 100장 |
| **(B) 창고 재고** | 현재 갖고 있는 양 | 30장 |
| **(C) 입고 예정** | 발주해서 오고 있는 양 | 20장 |
| **(D) 출고 예정** | 고객 주문 중 3일 후 출고분 | 10장 |
| **보유량** | B + C - D | 40장 |
| **발주량** | A - 보유량 | **60장** |

---

## 3. 리오더 대상 조건

### 3.1 포함 조건

리오더 대상이 되려면 **판매 효율(SPV)** 기준을 충족해야 합니다.

| 상품 유형 | 최소 SPV | 의미 |
|-----------|----------|------|
| 신상품 | 30% 이상 | 100명 중 30명 이상 구매 |
| 기존상품 | 65% 이상 | 100명 중 65명 이상 구매 |

> 💡 **SPV (Sales Per View)**: 상품 조회 대비 구매 비율

### 3.2 제외 조건

아래 조건 중 **하나라도 해당**되면 리오더에서 제외됩니다.

| 조건 | 기준 | 제외 이유 |
|------|------|----------|
| 낮은 등급 | E등급 또는 F등급 | 품질/판매 부진 상품 |
| 할인 이력 | 최근 60일 내 할인 | 재고 소진용 할인 상품 |
| 장기 재고 | 출시 70일 이상 + 추천 5장 | 수요가 낮은 상품 |

---

## 4. 차수별 발주 전략

### 4.1 차수란?

한 시즌 내에서 해당 상품이 **몇 번째 입고**인지를 의미합니다.

### 4.2 차수별 전략

| 구분 | 초도 ~ 4차 | 5차 이상 |
|------|-----------|----------|
| 상태 | 🔍 검증 중 | ✅ 검증 완료 |
| 예측 기간 | 2주 (14일) | 3주 (21일) |
| 최대 수량 | 10장 | 제한 없음 |
| 전략 | 조심스럽게 소량 | 과감하게 발주 |

### 4.3 상한 해제 조건

초도~4차라도 아래 조건 충족 시 **10장 제한이 해제**됩니다.

| 조건 | 기준 |
|------|------|
| 판매 효율 우수 | 카테고리 평균 대비 1.3배 초과 |
| 최근 판매 호조 | 최근 7일간 7장 이상 판매 |

---

## 5. 시즌 구분

차수 계산의 기준이 되는 시즌은 다음과 같이 구분됩니다.

| 현재 월 | 시즌 | 시즌 기간 |
|---------|------|----------|
| 9 ~ 12월 | F/W | 9월 1일 ~ 익년 2월 말 |
| 1 ~ 2월 | F/W (계속) | 전년 9월 1일 ~ 2월 말 |
| 3 ~ 8월 | S/S | 3월 1일 ~ 8월 말 |

---

## 6. 수요 예측 방법

### 6.1 예측 방식

| 방식 | 적용 조건 | 예측 방법 |
|------|----------|----------|
| 통계 예측 | 과거 판매 데이터 충분 | 유사 상품 패턴 분석 (Monte Carlo) |
| 휴리스틱 (고효율) | 데이터 부족 + 높은 SPV | 최근 7일 판매 × 2 |
| 휴리스틱 (적응형) | 그 외 | 최근 14일 평균 기반 |

### 6.2 유사 상품 그룹 기준

통계 예측 시 다음 조건이 같은 상품들을 그룹으로 묶어 분석합니다.

- 상품 유형 (신상품/기존상품)
- 카테고리
- 최근 7일 판매량 구간
- 최근 14일 판매량 구간

## 7. 전체 프로세스 요약

### 7.1 판단 흐름

```
전체 상품
    │
    ▼
┌─────────────────────────────┐
│ 1단계: SPV 기준 충족 여부    │
│    • 신상품: 30% 이상       │
│    • 기존상품: 65% 이상     │
└─────────────────────────────┘
    │ 미달 → ❌ 제외
    ▼
┌─────────────────────────────┐
│ 2단계: Price Down 여부      │
│    • E/F 등급 제외          │
│    • 60일 내 할인 이력 제외  │
└─────────────────────────────┘
    │ 해당 → ❌ 제외
    ▼
┌─────────────────────────────┐
│ 3단계: 차수 확인            │
│    • 초도~4차: 2주/10장     │
│    • 5차+: 3주/무제한       │
└─────────────────────────────┘
    │
    ▼
┌─────────────────────────────┐
│ 4단계: 발주량 계산          │
│    예상 판매량 - 보유량     │
└─────────────────────────────┘
    │
    ▼
  📦 발주 추천
```

### 7.2 핵심 원칙

| 원칙 | 설명 |
|------|------|
| 과재고 방지 우선 | 품절보다 과재고가 더 큰 손해 |
| 검증 후 확대 | 처음엔 소량, 잘 팔리면 늘림 |
| 악성재고 차단 | 할인/저등급 상품은 재발주 안 함 |

---

## 8. 용어 정리

| 용어 | 설명 |
|------|------|
| SPV | Sales Per View, 조회 대비 판매율 |
| 차수 | 시즌 내 입고 횟수 (초도, 2차, 3차...) |
| 리드타임 | 발주 후 입고까지 걸리는 시간 |
| Reserved | 주문 완료되었으나 아직 출고 전인 수량 |
| In-transit | 발주했으나 아직 입고되지 않은 수량 |
| Price Down | 할인 판매 (재고 소진 목적) |
